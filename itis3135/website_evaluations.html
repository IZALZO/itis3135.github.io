An Evalustion of Two Websites

This webpage is meant to scrutinize two websites based off of the C.R.A.P. principles.

Link to ARNGREN.net a Norwegian site that deals with the sale of many items.

  • This website adhears to contrast, although barely, most text is readable interms of color
  • This website does not adhear to repetiton as all elements are thrown across the screen in different sizes, the only form of repetition it utilizes is repeditally confusing its users.
  • This website does not adhear to alignment as again every element is spread wildly across all of its listings using no semblence of a grid.
  • This website does not adhear to proximity as as quite litterally every image is overlapping and some even cross the threshold of the main title.

Link to Amazon.com a general retail site.

  • This website adhears to contrast, all text is easily visible and readable even when backgrounds change.
  • This website adhears to repetiton, all navigation and heading are the same through all pages and every screen is similar in construct.
  • This website adhears to alignment, all listings and pictures are nicely aligned and use well spaced borders to form a concise grid.
  • This website adhears to proximity, all photos are well spaced though the navigation bar and header are a bit busy.
